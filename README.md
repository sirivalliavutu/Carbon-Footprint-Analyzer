# 🌍 Carbon Footprint Analyzer

A Python-based project that calculates carbon emissions generated from daily activities and helps users understand their environmental impact.

---

## ✨ Features

- Calculates electricity emissions
- Calculates transport emissions
- Food and waste impact analysis
- Carbon footprint categorization
- Environmental suggestions
- File handling for saving reports
- NumPy statistical calculations

---

## 🛠 Technologies Used

- Python
- NumPy
- OOP Concepts
- File Handling
- Exception Handling

---

## 📂 Project Structure

```bash
Carbon-Footprint-Analyzer/
│
├── main.py
├── carbon.py
├── suggestions.py
├── stats.py
├── results.txt
└── README.md
```

---

## ▶️ How to Run

```bash
python main.py
```

---

## 📊 Output

The program calculates:

- Electricity Emissions
- Transport Emissions
- Food Emissions
- Waste Emissions
- Total Carbon Footprint
- Trees Required to Absorb CO₂
